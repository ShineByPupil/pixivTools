# P站功能加强

## 一、收藏优化

- 鼠标**悬停**插画，显示 `快速收藏` 按钮
- 点击按钮后，**打开**并**跳转**到收藏页
- 修改收藏后**自动关闭**收藏页

## 二、添加收藏，自动填写标签

> 新增收藏时（修改无效），自动填写标签信息（需用户自行配置）

**设置弹窗（右下角）**

| 功能   | 说明                                   |
|------|--------------------------------------|
| 新增   | 增加一条标签配置                             |
| 标签   | 如果插画标签包含该字段，新增收藏时自动填入输入框             |
| 标签别名 | 可为空，自动填入使用别名。例如识别到 `カンナカムイ` 填写 `小肥龙` |
| 删除   | 删除该条标签配置                             |
